# Mood Compass 情緒紀錄與自我照護儀表板

GitHub repository name: `115504`

第 115504 組系統已整理為可操作的靜態前端工作台，包含互動任務、狀態管理與本機瀏覽器資料保存。

## 系統能力

- 記錄每日情緒、事件、照片與備註，形成可回顧的個人照護歷程。
- 整合待辦、相簿、音樂與成就，協助使用者建立穩定照護節奏。
- 提供親友可讀摘要，讓重要關係能看見情緒變化與支持需求。
- 以跨裝置網頁介面提供日常紀錄、統計與分享管理。
