# Mood Compass 情緒紀錄與自我照護儀表板

GitHub repository name: `115504`

此公開展示版依據第 115504 組學生文件的系統目的、功能與預期成果重新設計，重點是讓版型、視覺語言與操作情境更貼近該系統的產品定位。

## 展示重點

- 紀錄並統計每日情緒，讓使用者用更輕量的方式整理內在狀態。
- 結合待辦、交友、相簿、音樂播放與成就，建立溫和的自我照護節奏。
- 協助好友或家人理解情緒變化，降低不善表達者的溝通壓力。
- 以網站方式跨裝置使用，降低硬體與安裝門檻。
